# README MD

## web-python

Inserção no SQLite com frontend html + bootstrap  + flask + jinja e backend python

Desenvolvido com:

* python 3.9.6
* html

* bootstrap
* flask 2.0.1

* jinja2 3.0.1
* sqlite

Como usar:

Copie os arquivos para `<pasta>`

->cmd

cd `<pasta>`

* crie a tabela com criadbtb
* set FLASK_APP=student
* set FLASK_ENV=development
* set DEBUG=on
* flask run

-> browser

http://127.0.0.1:5000/
